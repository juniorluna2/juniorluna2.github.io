2022 Downriver Major Golf Pool

This site will be used to keep all participants up-to-date on the current Downriver Major Golf Pool.
Current Major: The Masters
Draft Start Date: April 5rd, 2022
Draft Start Time: 7:30pm
Number of Participants: 8
Pot Size: $80 (Winner takes pot!)

Rules:
Each Participant will select four (4) players from the list provided below. A random draft order will be conduct and order of draft will be a snake draft (ie. Round 1: 1 2 3 4, Round 2: 4 3 2 1 and so on...). The particiant with the lowest accumulative score will be the winner. Important: Each player that misses the cut will add ten (10) strokes to the participant overall's score. An example would be if the four (4) players selected, my accumulative score is -10 but two (2) players miss the cut, then my adjusted accumulative score would be +10.

I know it will be tough to keep track of everyone's selection so I will do my best to make all updates after each selection on this site as soon as the pick is in. For real-time updates, you may want to disable your 'cache' on your prefer web browser during this time.

Payment:
Ideally, if you can send payment in some sort of electronic form (PayPal, Zelle, Venmo), that would be great. If not, we can coordinate those details via text.
Winner will be paid out on Monday!!!

Ties:
There will be absolutely no ties!! After the draft is completed, using the same order as the draft, each participant will guess the winning score of the 2022 Masters. Par is 72 which a par winning score will be 288. If there is still a tie from here, we will complete a putt-off on a future date at Sandy Creek.
